# Assign to Buffer Elements

Assign to Buffer Elements

## Usage

``` r
# S3 method for class 'shard_buffer'
x[i, j, ...] <- value
```

## Arguments

- x:

  A shard_buffer object.

- i:

  Index or indices.

- j:

  Optional second index (for matrices).

- ...:

  Additional indices (for arrays).

- value:

  Values to assign.

## Value

The modified buffer (invisibly).
