# Scratch pool diagnostics

Scratch pool diagnostics

## Usage

``` r
scratch_diagnostics()
```

## Value

A list with counters and current pool bytes.

## Examples

``` r
scratch_diagnostics()
#> $hits
#> [1] 0
#> 
#> $misses
#> [1] 0
#> 
#> $bytes
#> [1] 0
#> 
#> $high_water
#> [1] 0
#> 
#> $max_bytes
#> [1] Inf
#> 
#> $needs_recycle
#> [1] FALSE
#> 
```
